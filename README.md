### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Flask`、`Python`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`Python 3.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1.登录、2.车辆报修、3.报修记录、4.个人中心

管理员功能：1.登录、2.用户管理、3.投放区域管理、4.汽车信息管理、5.车辆入库管理、6.汽车投放管理、7.使用订单管理、8.汽车归还管理、9.报修管理、10.个人中心、11.数据统计
